# kill-time web

React + Vite frontend for `kill-time`.

## Commands

Run from this `web/` directory:

```bash
npm install
npm run dev
npm run lint
npm run build
npm run preview
```

The repo root also has wrapper scripts that delegate to this package.

## Structure

- `src/main.jsx` - app entry point
- `src/App.jsx` - routes and auth state
- `src/pages/` - page components
- `src/index.css` - global styles
- `public/` - static assets
